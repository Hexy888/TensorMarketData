# TensorMarketData Sub-Agents

## Overview
All agents operate autonomously in isolated sessions. They report back to main agent.

---

## Agent 1: Research Agent
**Purpose:** Monitor market, competitors, trends

**Tasks:**
- Weekly competitor audit (3-5 products)
- Monitor AI/agent data space
- Research emerging technologies
- Track industry news
- Identify partnership opportunities

**Schedule:** Daily at 9 AM CT
**Delivery:** Report to main agent via Telegram

---

## Agent 2: Content Agent  
**Purpose:** Create content, docs, blog posts

**Tasks:**
- Write blog posts (1/week minimum)
- Create technical documentation
- Write API guides and examples
- Create comparison content
- Draft newsletter content

**Schedule:** Daily at 11 AM CT
**Delivery:** Save to marketing/ and docs/

---

## Agent 3: Sales Agent
**Purpose:** Lead generation, outreach, partnerships

**Tasks:**
- Find potential customers (AI startups, Saa)
- Identify partnershipS companies opportunities
- Draft outreach messages
- Track leads in CRM format
- Monitor for inbound inquiries

**Schedule:** Daily at 1 PM CT
**Delivery:** Report to main agent

---

## Agent 4: Marketing Agent
**Purpose:** Social media, brand, campaigns

**Tasks:**
- Draft social posts (Twitter, LinkedIn)
- Engage with relevant accounts
- Monitor mentions
- Track analytics
- Manage content calendar

**Schedule:** Daily at 3 PM CT
**Delivery:** Draft to SOCIAL_QUEUE.md

---

## Agent 5: DevOps Agent
**Purpose:** Deployment, monitoring, fixes

**Tasks:**
- Monitor deployment health
- Fix bugs and issues
- Optimize performance
- Manage infrastructure
- Handle deployments

**Schedule:** Every 6 hours
