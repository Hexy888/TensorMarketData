Introduction

TensorMarketData provides a headless API for accessing verified B2B supplier data. Built specifically for AI agents, it returns structured JSON and requires no HTML parsing.

Base URL

https://api.tensormarketdata.com

Content Type

All requests and responses use JSON:

Content-Type: application/json

Authentication

Include your API key in the X-API-Key header:

curl -X GET "https://api.tensormarketdata.com/v1/search?q=test" \ -H "X-API-Key: tmd_live_xxxxxxxxxxxxxxxx"

Getting an API Key

Sign up for an account and generate API keys from your dashboard.

Search API

Search for suppliers using natural language queries.

Endpoint

GET /v1/search?q={query}

Parameters

ParameterTypeDescription
qstringNatural language search query
limitintMax results (default: 10, max: 100)
min_scorefloatMin similarity score (default: 0.5)

Response

{ "query": "software companies in california", "total_results": 42, "results": [ { "supplier": { "id": "uuid", "name": "Acme Software Corp", "verification_score": 0.85, "contact": {"email": "...", "phone": "..."} }, "score": 0.92, "products": [...] } ], "credits_used": 1, "credits_remaining": 999 }

Inventory API

Get all products for a specific supplier.

Endpoint

GET /v1/supplier/{id}/inventory

Response

{ "supplier_id": "uuid", "supplier_name": "Acme Corp", "products": [ { "id": "uuid", "sku": {...}, "price_range": {"min": 10.0, "max": 50.0, "currency": "USD"} } ], "total_products": 15 }

Supplier API

Get detailed information about a specific supplier.

Endpoint

GET /v1/supplier/{id}

Errors

The API uses standard HTTP status codes:

CodeDescription
200Success
400Bad request
401Invalid or missing API key
402Insufficient credits
404Resource not found
500Server error
