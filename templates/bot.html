For Developers

Integrate the TensorMarketData API directly into your AI agents.

Python SDK

Quick Start

from tensormarket import TensorMarket bot = TensorMarket(api_key="your-key") # Natural language search results = bot.search("software companies in california") # Check inventory inventory = bot.get_inventory(supplier_id="uuid") # Subscribe to updates bot.webhooks.subscribe( url="https://your-agent.com/webhook", events=["supplier.updated", "product.new"] )

API Endpoints

GET /v1/search?q={query} GET /v1/supplier/{id}/inventory POST /v1/webhooks/subscribe
