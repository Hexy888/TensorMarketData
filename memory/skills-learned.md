- **How it works:**

  2. Generate search queries (pain points, competitor complaints, solution seeking)
  3. Search social media for matches
  4. Score leads (1-10) based on intent

- **What it does:** Enriches leads with emails, formats for Notion
- **Methods:**

  2. Instagram bio check
  3. Hunter.io API (free tier: 50 searches/mo)
  4. Email pattern guessing
- **Output:** JSON with domain, brand, contact email, confidence score
- **For us:** Enrich company data with verified emails
