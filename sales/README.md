# TensorMarketData - Sales Enablement Kit

## Overview

This folder contains all sales development materials for TensorMarketData's outbound efforts.

## Contents

| File | Description |
|------|-------------|
| `data-provider-partners.md` | Research on B2B data providers, brokers, and aggregators |
| `enterprise-customers.md` | Target AI agent companies and enterprise buyers |
| `cold-email-templates.md` | Personalized email templates for outreach |
| `pitch-deck-outline.md` | 15-slide partnership pitch deck structure |
| `follow-up-sequences.md` | Multi-touch cadences for different prospect types |
| `hashtags-communities.md` | Hashtags, LinkedIn groups, Slack communities, events |
| `crm-contact-lists.csv` | CRM-ready contact lists with companies and templates |

## Quick Start

### 1. Research Phase
- Review `data-provider-partners.md` for Tier 1-3 data providers
- Review `enterprise-customers.md` for AI agent and enterprise targets

### 2. Contact Research
- Use `crm-contact-lists.csv` as template
- Research specific contacts using LinkedIn, company websites
- Update CSV with verified contacts

### 3. Outreach
- Select appropriate template from `cold-email-templates.md`
- Personalize with company-specific details
- Use `follow-up-sequences.md` for cadences

### 4. Community Engagement
- Join communities listed in `hashtags-communities.md`
- Post on LinkedIn using recommended hashtags
- Attend relevant conferences

### 5. Partner Meetings
- Use `pitch-deck-outline.md` for presentations
- Customize for data provider vs. customer vs. enterprise
