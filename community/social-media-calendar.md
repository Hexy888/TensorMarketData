## Emergency Content (On-Call)

### Server Outage
1. **First 30 min:** Acknowledge on all platforms
2. **Update every 2 hours:** Progress updates
3. **Resolution:** Thank you + root cause summary

### Negative PR
1. **Assess:** Is it valid?
2. **Response:** Prepared statement (PR + Legal)
3. **Follow-up:** Detailed response within 24h

### Viral Moment
1. **Assess:** Is it positive?
2. **Engage:** Reply, thank, amplify
3. **Capture:** Screenshot metrics, testimonials
4. **Follow-up:** Press release, blog post
